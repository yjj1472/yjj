HTML이란 무엇인가

Hypertext mark up language (HTML) is the standard mark up language for creating web pages and web application. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages.

HTML describes the structure of a web page semantically and originally included cues for the appearance of documnent.

HTML elements are the building blocks of HTML pages.
